# T4 - EC2 & VPC Infrastructure using Terraform

## Objective

Provision AWS infrastructure using Terraform.

## Resources Created

* AWS VPC
* Public Subnet
* Internet Gateway
* Route Table
* Security Group
* EC2 Instance

## Terraform Commands Used

```bash
terraform init
terraform validate
terraform plan
terraform apply
```

## Outputs

* VPC ID
* Subnet ID
* Security Group ID
* EC2 Public IP
